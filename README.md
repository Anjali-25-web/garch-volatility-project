# 📊 Stock Price Volatility Forecasting using GARCH Model

## 📌 Project Overview
This project analyzes financial market volatility using statistical time series modeling (GARCH model).

## 📊 Dataset
Stock data is fetched using Yahoo Finance via Python (yfinance library).

## ⚙️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- ARCH (GARCH Model)
- Yahoo Finance API

## 📈 Methodology
- Log returns calculation
- Stationarity test (ADF test)
- GARCH(1,1) volatility modeling
- Forecasting future volatility

## 🎯 Outcome
- Captured volatility clustering in stock prices
- Forecasted market risk using statistical model

## 💡 Use Case
Used in risk management and financial analysis
